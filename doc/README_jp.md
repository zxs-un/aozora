「AOZORA」とは、ドイツ語で「所謂整然汎用計算機器」の意味で
「Allgemeine Ordentliche SOgenannte RechenenAnage」の頭字語です。

そして、「AOZORA」はまた「青空」のローマ字表記です。

この「アオゾラ」プロジェクトの目標は、学習や教育の為に、
74シリーズ標準ロジックICとブレッドボードを使用して、
自由で簡単なコンピューターを実装します。
